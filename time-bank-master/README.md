# 时间银行
这个项目写的很简单，主要是几个增删改查，主要就用了Vue + ElementUI + axios 
使用路由进行跳转
## 安装项目所需要的依赖
```
npm install
```

### 项目运行启动
```
npm run serve
```

### 项目打包
```
npm run build
```
